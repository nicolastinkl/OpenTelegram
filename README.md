# OpenTelegram
OpenTelegram for telegram android source code.

### 基于 Android telegram 源代码定制开发私密服务器版本的聊天工具，主要用于私密用户获取，密聊，私域转化，私密支付等使用场景。自定义转化渠道，用户进入 App后自动连接自动分配不同的客服对接，自动转发各种成功私密消息等
